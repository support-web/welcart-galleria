== 概要 ==

このデザインテーマは、Welcart 専用のネットショップテンプレートです。

レスポンシブWebデザインに対応しており、スマートフォン、タブレット、PCからお買い物ができるWelcart専用のテーマです。
WordPress3.4から追加された「テーマカスタマイザ—機能」にWelcart独自のオプション機能が搭載されており、
ライブプレビューを確認しながらリアルタイムに設定や変更が行なえます。
また、拡張プラグイン「WCEX」にも対応（一部除く）しております。


== インストール ==

1.管理パネルの[ 外観 ]＞[ テーマ ]ページ上部にある「新規追加」をクリックしてください。
2.テーマを追加ページ上部にある「テーマのアップロード」をクリックしてテーマの圧縮ファイルをそのままアップロードします。
3.アップロード完了後に該当テーマを「有効化」します。

同じテーマをアップロードする場合は、テーマに施した修正内容が全て消えてしまうのでご注意ください。
カスタマイズしたテーマはバックアップを取ってから削除・上書きを行ってください。


== ご利用上の注意点 ==

* WordPress 5.6・Welcart 2.6 以降の環境でご利用ください。
* 対応ブラウザはChrome、FireFox、Safari、Opera、Edgeです。
